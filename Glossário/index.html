Glossário das 8 pricipais tags HTML

title
O título do documento
meta
Informações sobre os dados da página HTML
head
Cabeçalho do documento, reune informações que não são apresentadas dentro da página
body
Representa o conteúdo do documento HTML
article
É um container para informações que podem ser distribuidas independente do restante do site
section
Container criado para uma seção do site, que contém elementos relacionados com o site
footer
Elemento que representa o rodapé do seu conteúdo
div
Container genérico para conteúdo de fluxo, que de certa forma, não representa nada.
